# 學貸計算機

這是一個練習 ReactJs 的專案，順便解決我計算學貸的問題


## 使用方式

計算需要的必備數值

1. 貸款金額
2. 貸款的學期數 (一學年兩學期)
3. 當前的利率 (2020年為因應防疫，皆為0.9%)


選填數值

1. 畢業第一年所還款的金額
2. 每次想還的本金金額
3. 輸入完畢後，下方的表格會顯示每次需要還款的金額
